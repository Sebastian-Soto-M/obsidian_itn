# Logical Link Control
```ad-note
title: Características

- Una subcapa independiente de la [[Capa de enlace de datos]],
la otra siendo [[Media Access Control|MAC]]
- Subcapa [[Institute of Electrical and Electronics Engineers|IEEE]] 802.2

```

```ad-summary
title: Funciones en [[Capa de enlace de datos|capa 2]]

- Se comunica entre **software** de red en **capas superiores**
y **hardware** del dispositivo en **capas inferiores**
- Coloca información para identificar el protocolo de [[Capa de red]] que se utiliza en la trama.
	- Permite que varios protocolos de capa 3 ([[IP versión 4|IPv4]], [[IP versión 6|IPv6]]) utilicen la misma interfaz de red y medios


```
